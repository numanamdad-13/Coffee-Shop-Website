# ☕ Coffee Shop Website

## 📖 Overview

Coffee Shop Website is a front-end web development project created as part of the AICT course. The website provides visitors with information about a coffee shop, including its menu, services, gallery, inventory, contact details, and online ordering.

---

## 🚀 Features

- Home Page
- About Us
- Services
- Coffee Menu
- Gallery
- Inventory
- Contact Us
- User Login
- User Registration
- Online Order
- Purchase Page
- Customer Feedback

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

Coffee-Shop-Website/

├── HTML Files

├── CSS Files

├── JavaScript

└── Images

---

## 🎯 Purpose

This project was developed for the AICT course to practice front-end web development concepts including page design, navigation, styling, and user interaction.
